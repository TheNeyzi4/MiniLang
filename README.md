# MiniLang

# English

A mini programming language implemented in <strong>C#</strong> with support for:

- Variables (`x = 10`)
- Basic operations (`+ - * /`)
- Print
- Simple if statements

**Example code:**

```
x = 10
y = 10
z = 25
print(z + x * y)
print(x - 5 + 28 * 156)
if x > 5 print(100 + 10)
```

**Output:**

```
125
4373
110
```
